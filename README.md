# MegaCap 50 — Beyond the Balance Sheet

An interactive data visualization dashboard exploring financial and non-financial metrics for the world's 50 largest publicly traded companies.

## Overview

This project provides a comprehensive look at the "MegaCap 50," going beyond traditional financial statements to explore the operational and strategic metrics that define corporate dominance in the modern era.

## Key Features

- **Interactive Dashboard**: Explore company rankings, sector breakdowns, and market capitalization at a glance.
- **Dynamic Visualizations**: View multi-year trends for key performance indicators (KPIs) using interactive charts powered by Chart.js.
- **Cross-Company Comparison**: Compare specific metrics (e.g., Cloud Revenue, R&D Spending, User Growth) across different companies and sectors.
- **Specialized Analysis Pages**:
    - **Economic Moats**: In-depth analysis of competitive advantages for tech giants.
    - **Dividend Intelligence**: Insights into US and HKEX dividend-paying companies.
    - **Value Investor's Compass**: Assessing intrinsic value and investment potential.
    - **S&P 500 Universe**: Broader market context.
- **Search & Filter**: Quickly find companies by name, ticker, or sector.
- **Responsive Design**: Optimized for both desktop and mobile viewing, featuring dark and light mode support.

## Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Charts**: [Chart.js](https://www.chartjs.org/)
- **Typography**: Google Fonts (Inter, JetBrains Mono)
- **Data**: JSON-based dataset (`data.json`)

## Project Structure

- `index.html`: The main entry point and landing page.
- `app.js`: Core application logic, including routing, search, filtering, and chart rendering.
- `base.css` & `style.css`: Modern, responsive styling with theme support.
- `data.js` & `data.json`: The underlying data repository for all 50 companies.
- `economic-moats.html`, `moat-analysis.html`: Deep-dive competitive analysis.
- `dividend-intelligence.html`, `hkex-dividend-intelligence.html`: Dividend-focused views.
- `value_compass.html`: Value investing assessment.
- `sp500.html`: S&P 500 reference page.

## Getting Started

Simply open `index.html` in any modern web browser to explore the dashboard.
